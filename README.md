## 仓库定位

- 这个仓库只存放科研绘图与展示资产，不承载训练或评测代码。
- 仍然直接用于主代码仓 README 展示的导出图，可以继续保留在对应代码仓中。
- 具体到每个项目的论文、社媒与引用信息，请进入各自子目录查看。
